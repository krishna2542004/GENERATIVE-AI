# AI Blog Generation App

Generate unique, creative blog posts instantly using Streamlit and the Groq API (Llama model).

## Features
- Instantly generate structured, context-aware blogs on custom topics
- Powered by Groq's Llama LLM
- Simple Streamlit interface
- Secure API key management via `.env`

## Getting Started

1. **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```

2. **Set up your `.env` file**
   - Copy `.env.example` to `.env`
   - Add your Groq API key

3. **Run the app**
    ```bash
    streamlit run blog_app.py
    ```

## Requirements

- Python 3.10+
- [Groq API key](https://groq.com/)
